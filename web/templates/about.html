What

secprac is a platform to create cyber security practice games for Linux systems similar to the Cyber Patriot competition.
secprac is free and open source software under the MIT License, originally developed by Theo Henson and Greer Page.

Why

secprac provides an easy to set up and extensible security training platform.
Geared to imitate Cyber Patriot, secprac is a complete and effective way to prepare for the competition.

How

Set up your own instance

Visit the wiki to see how you can create your own secprac practice game.

Connect to an instance

Provided you are on a virtual machine, run the following command in your terminal to install the client.
curl https://raw.githubusercontent.com/blueberry-jam/secprac-client/master/install.sh | sudo sh
Once the client is installed, run this command to start:
sudo secprac-start <user> <server url>
Replace <user> with the username of the current logged in user, and <server url> with the url of the serverâ€”e.g., the url of this website.
